Generated V & C monogram icons — do not hand-edit. To retune (colours, date,
lettering), edit `scripts/gen-icons.mjs` and run `node scripts/gen-icons.mjs`.

- `icon-192.png` / `icon-512.png` — manifest icons
- `icon-maskable-512.png` — Android maskable (extra safe-zone padding)
- `apple-touch-icon.png` — iOS home screen (180×180)
